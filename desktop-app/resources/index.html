Markdown Viewer Help

Application shortcuts

  • Use the view buttons in the toolbar to switch between Editor, Split, and Preview modes.
  • Sync scrolling is available in Split view to keep the editor and preview aligned.
  • Import, Export, Copy, Share, and Theme toggle actions are always available in the header toolbar.

Toolbar descriptions

  • Undo/Redo and Clear Formatting help you manage editing changes quickly.
  • Text styling tools cover bold, italic, strikethrough, quotes, headings, and list formatting.
  • Insert helpers add links, images, code blocks, tables, emojis, symbols, and alerts.
  • Use Fullscreen, Find & Replace, Help, and About Markdown for focused editing.

Markdown tips

  • Create headings with #, ##, and ### prefixes.
  • Emphasize text with **bold**, *italic*, or ~~strikethrough~~.
  • Use - or 1. to build lists, and triple backticks for code blocks.

Keyboard shortcuts

  • Ctrl/⌘ + Z — Undo
  • Ctrl/⌘ + Shift + Z — Redo
  • Ctrl/⌘ + F — Open Find & Replace
  • Ctrl/⌘ + C/V — Copy/Paste
